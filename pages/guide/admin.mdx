# admin & setup

configure your server. set up automation. manage systems.

<Callout type="caution">
  these commands require `ADMINISTRATOR` or `MANAGE_GUILD` permissions.
</Callout>

---

## server hub

### `/admin`

central hub for server configuration, AI settings, and logging.

**usage:** opens interactive menu

#### hub options

| category            | description                  |
| ------------------- | ---------------------------- |
| **server settings** | updates channel, staff roles |
| **mina ai**         | ai responder configuration   |
| **logging**         | audit log setup              |
| **status**          | view all current settings    |

#### server settings

| option              | description                               |
| ------------------- | ----------------------------------------- |
| set updates channel | channel for bot updates and announcements |
| add staff role      | add a role to the staff list              |
| remove staff role   | remove a role from staff list             |

#### mina ai

| option                | description                                 |
| --------------------- | ------------------------------------------- |
| toggle ai             | enable/disable ai responder for this server |
| set free-will channel | channel where i respond without @mention    |
| toggle mention-only   | when on, i only respond to @mentions        |
| manage free-will      | add/remove free-will channels               |

#### logging (page 1)

| option               | description                   |
| -------------------- | ----------------------------- |
| set log channel      | channel for all audit logs    |
| toggle all logging   | master switch for logging     |
| message edits        | log when messages are edited  |
| message deletes      | log when messages are deleted |
| member role changes  | log when member roles change  |
| ghost ping detection | log when someone ghost pings  |

#### logging (page 2)

| option           | description                      |
| ---------------- | -------------------------------- |
| all channel logs | toggle all channel event logging |
| channel created  | log new channel creation         |
| channel updated  | log channel modifications        |
| channel deleted  | log channel deletions            |
| all role logs    | toggle all role event logging    |
| role created     | log new role creation            |
| role updated     | log role modifications           |
| role deleted     | log role deletions               |

| ----------------- | ----------------------------- | --------- |

### `/anti`

toggle protection features.

| subcommand    | description                    |
| ------------- | ------------------------------ |
| `ghostping`   | toggle ghost ping detection    |
| `spam`        | toggle spam protection         |
| `massmention` | toggle mass mention protection |

**example:**

```
/anti ghostping
/anti spam
```

---

### `/autodelete`

auto-remove messages matching criteria.

| subcommand    | description               | options           |
| ------------- | ------------------------- | ----------------- |
| `attachments` | auto-delete attachments   | `status` (on/off) |
| `invites`     | auto-delete invite links  | `status` (on/off) |
| `links`       | auto-delete links         | `status` (on/off) |
| `maxlines`    | auto-delete long messages | `amount` (number) |

**example:**

```
/autodelete invites status:on
/autodelete maxlines amount:20
```

### `/roles`

bulk role management hub.

**usage:** opens interactive menu

**required permissions:** `Manage Guild`, `Manage Roles`

#### operations

| operation       | description                          |
| --------------- | ------------------------------------ |
| **cleanup**     | bulk delete roles by criteria        |
| **create role** | create new role with custom settings |
| **autorole**    | auto-assign role to new members      |
| **add to user** | assign roles to multiple users       |

#### cleanup methods

| method         | description                           |
| -------------- | ------------------------------------- |
| empty roles    | delete roles with no members          |
| name prefix    | delete roles starting with prefix     |
| below position | delete roles below hierarchy position |
| older than     | delete roles created before date      |

#### autorole

| action        | description                 |
| ------------- | --------------------------- |
| enable/change | select role for new members |
| disable       | turn off autorole           |

**requirements:**

- role must be below my highest role
- role cannot be managed by integration
- role cannot be @everyone

## communication

### `/say`

speak through the bot.

| option       | description               | required |
| ------------ | ------------------------- | -------- |
| `message`    | message content           | yes      |
| `channel`    | target channel            | no       |
| `message_id` | message to reply to       | no       |
| `edit`       | edit existing bot message | no       |
| `ping`       | allow mentions            | no       |

**example:**

```
/say message:hello everyone! channel:#announcements
/say message:updated content edit:123456789
```
