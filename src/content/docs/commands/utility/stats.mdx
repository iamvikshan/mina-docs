## Stat Command Central

<CardGrid>
  <Card title="🛠️ Admin Setup" icon="seti:tools">
    - `!statstracking <on|off>` - Flip the switch on stat tracking
    - `!levelup message <new-message>` - Customize those level-up cheers!
    - `!levelup channel <#channel>` - Choose where to shout about achievements
  </Card>

  <Card title="📊 User Insights" icon="seti:graph">
    - `/stats` - How awesome are you? Let's find out!
    - `/rank` - Where do you stand in the server hierarchy?
  </Card>
</CardGrid>
