## games

### `/tictactoe`

| option | description | required |
|--------|-------------|----------|
| `user` | user to challenge | yes |

---

### `/hangman`

play a word guessing game.

| option | description | required |
|--------|-------------|----------|
| `theme` | word category | yes |

**themes:** nature, sport, color, camp, fruit, discord, winter, pokemon

---

### `/tod`

play truth or dare and other party games.

| subcommand | description | options |
|------------|-------------|---------|
| `truth` | get a truth question | `rating` (optional) |
| `dare` | get a dare challenge | `rating` (optional) |
| `paranoia` | get a paranoia question | `rating` (optional) |
| `nhie` | never have i ever | `rating` (optional) |
| `wyr` | would you rather | `rating` (optional) |
| `hye` | have you ever | `rating` (optional) |
| `wwyd` | what would you do | `rating` (optional) |
| `random` | random category | `rating` (optional) |

**ratings:** PG (default, 13+), PG-16 (16+), R (18+ NSFW only)

### `/image`

| subcommand | description | options |
|------------|-------------|---------|
| `animal` | random animal image | `name` |
| `anime` | random anime image | `name` |

**animals:** cat, dog, panda, fox, red_panda, koala, bird, raccoon, kangaroo

**anime:** waifu, shinobu, megumin
