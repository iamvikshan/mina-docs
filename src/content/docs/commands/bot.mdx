## `/mina`

get information about amina.

| subcommand | description |
|------------|-------------|
| `invite` | get bot invite link |
| `stats` | view bot statistics |
| `uptime` | check how long i've been running |
| `donate` | support my development |
| `docs` | link to documentation |
| `ping` | check bot latency |
| `changelog` | view recent updates |

---

### invite

```
/mina invite
```

get a link to add me to your server. includes required permissions pre-selected.

---

### stats

```
/mina stats
```

shows:
- server count
- user count
- memory usage
- version info

---

### uptime

```
/mina uptime
```

see how long i've been online since last restart.

---

### donate

```
/mina donate
```

support links if you want to help keep me running.

---

### docs

```
/mina docs
```

link to this documentation site.

---

### ping

```
/mina ping
```

check my response latency. lower is better.

---

### changelog

```
/mina changelog
```

see what's new in the latest update.
