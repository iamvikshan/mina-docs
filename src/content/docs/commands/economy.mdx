## banking

### `/bank`

manage your virtual wallet.

| subcommand | description | options |
|------------|-------------|---------|
| `balance` | view wallet and bank balance | `user` (optional) |
| `deposit` | move coins from wallet to bank | `coins` |
| `withdraw` | take coins from bank to wallet | `coins` |
| `transfer` | send coins to another user | `user`, `coins` |

**example:**
```
/bank balance
/bank deposit coins:1000
/bank transfer user:@friend coins:500
```

**why use bank?**
- separate savings from spending
- track wealth better

## gambling

### `/gamble`

risk your coins on a slot machine spin.

| option | description | required |
|--------|-------------|----------|
| `coins` | number of coins to bet (minimum 10) | yes |

**example:**
```
/gamble coins:100
```
