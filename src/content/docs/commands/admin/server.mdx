## 1. Channel Counters
Make dynamic channels that show your server stats!

```
/counter type:[USERS|MEMBERS|BOTS] name:"Server Members: {count}"
```

Available counter types:
- `USERS` - Total server count (members + bots)
- `MEMBERS` - Human members count
- `BOTS` - Bot count

## 2. ⚠️ Warning System

Set up automated punishments for repeated warnings:
```
/maxwarn limit amount:[number]
/maxwarn action [timeout|kick|ban]
```

## 3. 📝 Logging

Keep track of all moderation actions:
```
/logs channel:#channel
/logs off
```

## 5. Role Management

### Auto Role
Automatically give roles to new members:
```
/autorole role:@Role
/autorole off
```

### Reaction Roles
Let members self-assign roles:
```
/addrr channel:#channel messageId:123456 role:@Role emoji:👍
/removerr channel:#channel messageId:123456
```
