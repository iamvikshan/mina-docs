## Giveaway Command Central

<CardGrid>
  <Card title="🚀 Launch Party" icon="seti:rocket">
    - `/giveaway start <#channel>` - Kick off the celebration!
  </Card>

  <Card title="⏯️ Party Control" icon="seti:play">
    - `/giveaway pause <messageId>` - Need a quick intermission?
    - `/giveaway resume <messageId>` - The party must go on!
    - `/giveaway end <messageId>` - All good things must come to an end
  </Card>

  <Card title="🎭 Plot Twist" icon="seti:tree">
    - `/giveaway reroll <messageId>` - Oops! Let's pick another winner!
  </Card>

  <Card title="📋 Party Planner" icon="seti:clipboard">
    - `/giveaway list` - Check out all the ongoing fun!
    - `/giveaway edit <messageId>` - Tweak the party plans on the fly
  </Card>
</CardGrid>

## How to Party 101

<Tabs>
  <TabItem label="Start the Fun">
    ```
    /giveaway start #giveaway-channel
    ```

  <TabItem label="Keep it Fresh">
