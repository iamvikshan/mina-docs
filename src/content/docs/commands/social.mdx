## invite tracking

### `/invite`

track invites and manage invite rewards.

| subcommand | description | options |
|------------|-------------|---------|
| `view` | check invite count | `user` (optional) |
| `codes` | list invite codes | `user` (optional) |
| `inviter` | see who invited a user | `user` (optional) |
| `ranks` | display invite rank rewards | - |
| `reset` | clear invites for user (admin) | `user` |
| `import` | sync existing invites (admin) | `user` (optional) |
| `tracking` | enable/disable tracking (admin) | `status` |

**example:**
```
/invite view
/invite codes user:@recruiter
/invite inviter user:@newmember
```

---

### invite ranks

reward members who bring in new users.

| subcommand | description | options |
|------------|-------------|---------|
| `rank add` | create invite rank reward | `role`, `invites` |
| `rank remove` | delete invite rank | `role` |

**example:**
```
/invite rank add role:@Recruiter invites:10
/invite rank remove role:@Recruiter
```

<Aside type="tip">
use `/invite ranks` to see all configured rank rewards.
</Aside>
