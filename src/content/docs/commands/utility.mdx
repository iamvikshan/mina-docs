## quick add

### `/add`

quick add for reminders, colors, and invites.

| subcommand | description | options |
|------------|-------------|---------|
| `reminder` | set a reminder | `message`, `in` (optional, default 6h) |
| `color` | create color option (admin) | `name`, `hex` |
| `invites` | add invites to user (admin) | `user`, `invites` |

**example:**
```
/add reminder message:check the oven in:30m
/add color name:sunset hex:#FF6B35
/add invites user:@member invites:10
```

## status

### `/afk`

set an AFK status. mentions will notify you when you return.

| option | description | required |
|--------|-------------|----------|
| `reason` | why you are going afk | yes |
| `duration` | duration in minutes (1-43200) | no |

**example:**
```
/afk reason:eating dinner
/afk reason:sleeping duration:480
```

## colors

### `/color`

pick a custom name color for yourself.

| subcommand | description | options |
|------------|-------------|---------|
| `set` | apply a color role | `color` |
| `remove` | remove your color | - |
| `list` | show available colors | - |
| `delete` | remove from palette (admin) | `name` |

**example:**
```
/color list
/color set color:sunset
/color remove
```

## profiles

### `/profile`

customize your bio, pronouns, and more.

**usage:** opens interactive profile editor

<Tabs>
  <TabItem label="operations">
    | operation | description |
    |-----------|-------------|
    | **edit profile** | update your profile information |
    | **privacy settings** | control what others can see |
    | **clear profile** | start fresh with a blank canvas |
  </TabItem>
  
  <TabItem label="edit profile">
    | category | fields |
    |----------|--------|
    | basic info | pronouns, birthdate, region, languages, timezone |
    | misc info | bio, interests, socials, favorites, goals |
    
    **formats:**
    - birthdate: DD/MM/YYYY or MM/YYYY
    - languages: comma-separated list
    - socials: platform:handle (e.g., twitter:username)
  </TabItem>
  
  <TabItem label="privacy">
    toggle visibility for:
    - age
    - region
    - birthdate
    - pronouns
  </TabItem>
</Tabs>

**tip:** use `/info profile` to view your profile or someone else's.

## reminders

### `/reminder`

open the reminder management hub.

**usage:** opens interactive hub

| operation | description |
|-----------|-------------|
| **view reminders** | see all your active reminders |
| **clear all** | delete all reminders (if any exist) |

**notes:**
- maximum 25 reminders per user
- shows next upcoming reminder with relative time
- set your timezone in `/profile` for accurate times

**tip:** use `/add reminder` for quick creation.
