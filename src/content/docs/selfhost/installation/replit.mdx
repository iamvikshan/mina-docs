## Fork Repository

<Steps>

1. Fork the repository:
   - Option A: Fork directly on [Replit](https://replit.com/@vikshan/amina)
   - Option B: Fork on [GitHub](https://github.com/vixshan/amina/fork) , then import to Replit
     1. Click on the `Import from GitHub` button in Replit
     2. Paste the URL of your forked repository

</Steps>

## Run Commands

There are two ways to run the bot on Replit:

<Tabs>
  <TabItem label="Terminal Command">
    Run the bot by typing one of these commands in the terminal:
    ```bash
    node .
    ```
    Or
    ```bash
    npm run start-replit
    ```
  </TabItem>
  <TabItem label="Start Button">
    To use the start button:
    1. While forking the repl, paste one of the above commands in the start command box.
    2. Or modify the `package.json` file:
    ```json
    // package.json
    "start": "node bot.js",
    ```
    <Aside type="caution">
      If you modify `package.json`, use `git update-index --assume-unchanged package.json` to avoid committing this change to GitHub. Always adjust the start script based on your environment if you do commit changes.
    </Aside>
  </TabItem>
</Tabs>
