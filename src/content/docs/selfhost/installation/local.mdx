## Installation Steps

<Steps>

1. Clone the Repository
   ```bash
   git clone https://github.com/vixshan/amina.git
   cd amina
   ```

2. Install Dependencies
   ```bash
   npm install
   ```

3. Set Up Environment Variables

   Rename `.env.example` to `.env and` and fill in the required information.

4. Start the Bot
   ```bash
   npm start
   ```

5. For development purposes, you can use the following command to start the bot with nodemon:
   
   <PackageManagers type="run" args="dev" />

</Steps>

## Environment Variables

<Aside type="caution">
  Make sure to keep your `.env` file secure and never share sensitive information.
</Aside>

Here's an example of what your .env file should look like:

```bash
// .env
# Bot Token [Required]
BOT_TOKEN=

# Mongo Database Connection String [Required]
MONGO_CONNECTION=

# Webhooks [Optional]
ERROR_LOGS=
JOIN_LEAVE_LOGS=
FEEDBACK_URL=

# Dashboard [Required for dashboard]
CLIENT_SECRET=
SESSION_PASSWORD=

# Required for Weather Command (https://weatherstack.com)
WEATHERSTACK_KEY=

# Required for image commands (https://strangeapi.fun/docs)
STRANGE_API_KEY=

# SPOTFIY [Required for Spotify Support]
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=

# Required for OpenAI (https://beta.openai.com/)
OPENAI=
```
