# Amina Dev Portal

Welcome to the Amina Developer Portal — the central hub for API documentation and developer resources for the [Amina Discord Bot](https://4mina.app).

## What's here?

### API Reference

Complete API documentation for [api.4mina.app](https://api.4mina.app), including:

- **Cards** — Generate rank cards, welcome cards, Spotify cards
- **Filters** — Apply image filters (greyscale, blur, sepia, etc.)
- **Overlays** — Add overlays (approved, rejected, wasted, etc.)
- **Generators** — Create memes (drake, change my mind, facts, etc.)
- **Utilities** — Color images, circle crops, and more
- **Bots** — Bot discovery and statistics

### Developer Wiki

Guides and documentation for extending, deploying, and understanding Amina's internals:

- System architecture and command handling
- AI integration (Mina AI)
- Moderation, economy, music, and more
- Self-hosting and deployment guides
